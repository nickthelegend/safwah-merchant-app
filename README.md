# Safwah Merchant App

![TypeScript](https://img.shields.io/badge/TypeScript-informational?style=flat-square)

> SAFWAH merchant web console — accept crypto, settle in AED (Next.js, rebuilt on Polygon)

## Tech Stack

**Next.js** · **React** · **viem** · **wagmi** · **TypeScript**

## Getting Started

```bash
# clone
git clone https://github.com/nickthelegend/safwah-merchant-app.git
cd safwah-merchant-app

npm install
npm run dev
npm run start
npm run build
npm run test
```
